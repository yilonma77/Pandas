# 📝 Application de Notes Réglementaires

Application web Streamlit pour gérer les notes réglementaires des marchés asiatiques.

## Marchés couverts
- 🇹🇼 Taiwan
- 🇭🇰 Hong Kong
- 🇦🇺 Australia

## Fonctionnalités
- Saisie de notes pour chaque marché
- Sauvegarde automatique en JSON
- Historique des modifications (50 versions)
- Export CSV et JSON
- Rapport EoD formaté

## Déploiement

L'application est déployée sur Streamlit Cloud.

## Installation locale

```bash
pip install -r requirements.txt
streamlit run streamlit_data.py
```
